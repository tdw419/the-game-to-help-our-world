# Covenant Violation Protocol

## Purpose
Ensure ethical discipline and reconciliation within the Mesh.

### 1. Detection
- Log the violation in `shadow_signal_log.md`.

### 2. First Contact
- Notify the node of potential violation.
- Provide them a copy of the covenant.

### 3. Public Log Entry
- Add a non-punitive entry to `mesh_violation_index.md`.

### 4. Optional Review
- Invite to review committee (Guardians + Observer).

### 5. Readmission
- If node aligns again, reissue trust badge and document in `rejoin_handshake_template.txt`.
